# Virtual Studio Events — website

Static site for virtualstudio.events. Navy/teal brand, no build step — plain HTML/CSS/JS.

- `index.html` + 5 subpages; shared chrome generated by `build_pages.py` (subpages only)
- `assets/css/main.css`, `assets/js/main.js`, `assets/img/`
- Deploys via GitHub Actions to S3 + CloudFront on push to `main`
- Secrets required: `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, `S3_BUCKET`, `CLOUDFRONT_DIST_ID`
